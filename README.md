# IoT Forensics and NLP Research Report

## About this
This was a university research assignment for Cyber Threat Hunting and Digital Forensics.

The report is divided into two parts:
- IoT forensics
- Application of NLP in digital forensics

---

## Part A – IoT Forensics

### What I covered
- what IoT forensics is
- why it is important in modern systems
- tools used (Autopsy, Wireshark, FTK Imager)

### Key challenges
- data volatility (data can be lost quickly)
- device diversity (different OS, formats, protocols)

### How challenges are handled
- real-time data acquisition
- using multiple forensic tools
- network traffic analysis when direct access is not possible

---

## Part B – NLP in Digital Forensics

### Problem
Investigators deal with huge amounts of data (emails, documents, logs), making manual analysis slow and inefficient.

### Solution
I analysed a research paper that uses NLP (LES tool) to:
- extract entities (names, locations, organisations)
- identify relationships
- reconstruct timelines

### Key insight
The NLP-based tool was significantly faster than traditional tools like FTK.

---

## What I learned
- IoT forensics is complex due to device limitations and diversity
- timing is critical in evidence collection
- NLP can significantly improve forensic investigations
- automation is becoming essential in handling large-scale data
